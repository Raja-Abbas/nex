# AnythingLLM on Railway is here!
<div className="left">
&Prime;<span>&#9888;</span> Heads up! Due to Railway container permissions, web scraping of any sort is disabled! You will need to upload those site's HTML manually.&Prime;
</div>
<br/>
**A full-stack application that enables you to turn any document, resource, or piece of content into context that any LLM can use as references during chatting. This application allows you to pick and choose which LLM or Vector Database you want to use as well as supporting multi-user management and permissions.**
<br/>
**This is a fully imaged version of the primary repo!**

**Includes all features of the main app including**
<ul className="">
- **Private embeddings**
- **LLM Support**
- **Built-in vector database**
- **External chatbot support**
- **Multi-user authentication**
- **Admin controls**
</ul>

## Supported LLMS ##

<ul className="">
- **OpenAI**
- **Azure OpenAI**
- **Anthropic ClaudeV2**
- **Google Gemini Pro**
- **Ollama (mac/i model)**
- **LLM Suite (all models)**
- **LocalAI (all models)**
- **Together AI (chat models)**
- **Mistral**
</ul>

## Supported Embedding modules ##

<ul className="">
- **Native Embedder (default)**
- **OpenAI**
- **Azure OpenAI**
- **LLM Suite (all)**
- **LocalAI (all)**
</ul>

## Supported Vector Databases ##

<ul className="">
- **LanceDB (default)**
- **Atlas DB**
- **Pinecone**
- **Weaviate**
- **Qdrant**
- **Milvus**
- **Zilliz**
</ul>

***Note:*** **This image is updated often but is not always in sync with the latest tag on Docker for the master branch. If something is missing ping us on Discord!**

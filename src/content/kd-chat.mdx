K-d Chat on NexLayer is here!
===============================

**Kd-Chat** is a straightforward yet powerful web application built with ReactJS and Express, designed to query Ollama's Dolphin-Phi model. This app serves as a user-friendly interface that allows developers and users to interact with the Dolphin-Phi model seamlessly, making it easy to explore the model's capabilities. By leveraging ReactJS for the frontend and Express for the backend, Kd-Chat offers a lightweight solution that can be easily integrated into various development workflows.

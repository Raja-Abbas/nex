**Strapi Community Edition** is a free and open-source headless CMS enabling you to manage any content, anywhere.

## Key Features

- **Self-hosted or Cloud:** Host and scale Strapi projects your way. Deploy to Strapi Cloud or to platforms like AWS, Azure, Google Cloud, or DigitalOcean.
- **Modern Admin Panel:** Elegant, customizable, and fully extensible.
- **Multi-database Support:** Choose your preferred database: PostgreSQL, MySQL, MariaDB, or SQLite.
- **Customizable:** Fully customize APIs, routes, or plugins to fit your needs.
- **Blazing Fast and Robust:** Built on Node.js and TypeScript for reliable performance.
- **Front-end Agnostic:** Use any front-end framework (React, Next.js, Vue, Angular, etc.), mobile apps, or even IoT.
- **Secure by Default:** Includes policies for CORS, CSP, P3P, Xframe, XSS, and more.
- **Powerful CLI:** Scaffold projects and APIs on the fly.

## Database Compatibility

| Database       | Recommended | Minimum |
| -------------- | ----------- | ------- |
| **MySQL**      | 8.0         | 5.7.8   |
| **MariaDB**    | 10.6        | 10.3    |
| **PostgreSQL** | 14.0        | 11.0    |
| **SQLite**     | 3           | 3       |

## Additional Features

- **Content Types Builder:** Create flexible publishing experiences with fields, components, and Dynamic Zones.
- **Media Library:** Manage images, videos, audio, or documents with ease.
- **Internationalization:** Manage and distribute localized content in multiple languages.
- **Role-Based Access Control:** Create custom roles and permissions for admin and end users.
- **GraphQL or REST:** Consume the API using REST or GraphQL.
